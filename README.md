## 📌 About This Project

Welcome to the **Laravel Learning Lab** — a personal sandbox where I’m mastering Laravel from the ground up. This project explores key Laravel concepts through real implementations, clean code, and best practices.

🔧 Ideal for backend development, REST APIs, authentication, templating, and dynamic full-stack projects.

---

## 🧠 Key Topics I'm Exploring

### 🔹 Core Laravel Features
- Laravel installation & environment setup
- Routing & controllers
- Blade templating engine
- Eloquent ORM & migrations
- Form validation & request lifecycle
- Authentication & middleware
- Artisan CLI & environment management

### 🔹 Real App Concepts
- Reusable layouts with Blade
- CRUD operations with database
- RESTful API setup (optional)
- MVC architecture in practice
- UI integration (Bootstrap/Tailwind optional)

## 🛠️ Setup Instructions

### ▶️ Run Locally
```bash
git clone https://github.com/Matrixxboy/portfolio.git
cd portfolio
composer install
cp .env.example .env
php artisan key:generate
php artisan serve
```
