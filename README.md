## 📌 About

**PHP Learning Journey** is a personal repository to master core PHP concepts and real-world applications. It lays the groundwork for backend development and Laravel. The project includes practical examples, clean code, and best practices to build a solid understanding of PHP programming.

---

## 🧠 What You'll Learn

- 🔹 PHP syntax, variables & data types
- 🔹 Control structures (if, else, loops, switch)
- 🔹 Functions & scope
- 🔹 Arrays, strings, superglobals
- 🔹 Form handling & file operations
